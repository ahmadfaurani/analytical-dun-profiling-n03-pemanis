## Key Findings (BLUF)

**Seat Status:** **BN-Favored (Comfortable but Not Safe)**

N03 Pemanis is a BN-leaning swing seat where the incumbent holds structural advantages (63.3% Malay electorate + incumbency + low-turnout machinery), but PH retains an upset path if Chinese turnout exceeds 75% and Malay vote split compresses BN margin. PN-MIPP plays spoiler role; even 1,500-2,000 votes from BN could make this competitive.

**2026 Projections (Baseline: 60-65% Turnout):**
- **BN:** 52-55% (9,500-10,200 votes) — Favorite
- **PH:** 37-40% (6,800-7,500 votes) — Underdog
- **PN-MIPP:** 7-9% (1,200-1,800 votes) — Spoiler
- **BN Margin:** 2,000-2,700 votes (comfortable but not safe)

**Decisive Variables:**
1. Turnout % (<60% = BN safe, >75% = PH competitive)
2. Malay vote split (BN needs 70%+, PN >25% compresses margin)
3. Chinese consolidation (PH needs 75%+ turnout + 85%+ vote share)
